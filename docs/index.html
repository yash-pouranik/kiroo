Capabilities

Everything You Need To
Master Your APIs

From zero-effort recording to AI-powered impact analysis, Kiroo replaces 6 tools with one CLI.

🕰️

Time-Travel Proxy

Start a proxy, click around your app — every API call is recorded automatically without typing commands.

🧠

AI Blast Radius

Let AI explain which pages will break and why — before you ship breaking changes to production.

📸

Snapshots & Diffing

Save your API state at any point. Compare versions with deep structural analysis that catches silent renames.

📄

Auto OpenAPI Specs

Generate a full Swagger/OpenAPI 3.0 specification from recorded traffic. Zero YAML writing.

🔐

Secret Redaction

Passwords, tokens, and API keys are masked at save time. Your .kiroo/ folder is Git-safe by default.

🧪

Zero-Code Testing

Validate responses in one line. No test framework needed — just status codes and field assertions.

🚀

Load Benchmarking

Stress test any endpoint with configurable concurrency. Find your latency limits instantly.

🌍

Variable Chaining

Save tokens from responses and inject them into future requests. Auth flows in two commands.

One CLI. Infinite Power.

~/my-project — kiroo
$ kiroo proxy --target http://localhost:3000
📡 Kiroo Proxy Started
Listening on : http://localhost:8080
Forwarding to: http://localhost:3000
 
POST /api/login 200 — 142ms
GET /api/users 200 — 89ms
GET /api/users/5 404 — 12ms

$ kiroo snapshot save v1-stable
📸 Snapshot saved: v1-stable (4 interactions)
 
$ kiroo analyze v1-stable v2-refactor --ai
🚨 HIGH GET /api/users
Field user_id removed → userId (rename candidate)
🧠 AI: Frontend profile pages will break. Update all references to user_id.
 
$ kiroo export --format openapi
✅ Export successful (openapi) — openapi.json

Quick Start

Up and Running in 30 Seconds

1

Install

$ npm install -g kiroo
2

Initialize

$ kiroo init
3

Record

$ kiroo proxy --target :3000
4

Analyze

$ kiroo analyze v1 v2 --ai

Commands

15+ Tools, One CLI

kiroo get|post|put|delete <url>Execute and record API requests
kiroo proxy --target <url>Auto-capture frontend ↔ backend traffic
kiroo snapshot save|compareVersion and diff your API state
kiroo analyze <v1> <v2> --aiAI-powered semantic blast radius
kiroo export --format openapiAuto-generate OpenAPI 3.0 specs
kiroo check <url> --status 200Zero-code API testing
kiroo bench <url> -n 100 -c 10Load testing with concurrency
kiroo replay | edit | import | scrubReplay, edit, import cURL, redact secrets
